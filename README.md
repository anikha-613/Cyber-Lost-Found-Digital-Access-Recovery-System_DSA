Cyber Lost & Found – Digital Access Recovery System
Project Description

Cyber Lost & Found is a Java-based backend system designed to help students recover passwords for their digital campus services such as ERP, LMS, Library, and WiFi.

The system reduces the manual effort of administrators by organizing password recovery requests using Data Structures and Algorithms (DSA).

This project is implemented as a terminal-based application that runs through the Command Prompt.

Technologies Used

Java

Object-Oriented Programming (OOP)

Command Prompt (CLI Interface)

Data Structures Used

This project demonstrates the use of important DSA concepts:

Linked List

Used to store and manage student records dynamically.

Queue

Used to manage password recovery requests in First-In-First-Out (FIFO) order.

Stack

Used to maintain password history for accounts.

Binary Search

Used to search for students efficiently using their roll number.

Features
Student Portal

Students can:

Register using a 10-digit roll number and password

Add digital accounts such as:

ERP

LMS

Library

WiFi

Store passwords for their accounts

View all saved accounts

Request password recovery if they forget their credentials

Add additional accounts when required

Admin Portal

Administrators can:

Login using a 5-digit admin ID and password

View all password recovery requests

Process recovery requests using a Queue

Identify which student requested password recovery

Security Rules

The system enforces basic validation:

Roll number must contain exactly 10 digits

Admin ID must contain 5 digits

Password must:

Be at least 6 characters long

Contain both letters and numbers
